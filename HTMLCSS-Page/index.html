Vorbereitung

Es sollte bei jedem CSS-Dokument zu Beginn die folgenden Zeilen eingefügt werden. Ohne diese Zeilen dehnt ein eingefügtes Padding die Größe der Box nach außen hin aus. Gerade weil diese Funktionalität nur optional eingesetzt wird, sollte diese Verzerrung immer im Hinterkopf sein und beachtet werden.

html {
box-sizing: border-box;
}
*, *:before, *:after {
box-sizing: inherit;
}

Das Fundament

Als Basis haben wir eine einfache Box. Diese Box kann verschiedene Eigenschaften haben. Mindestens aber besitzt Sie die Eigenschaften "height" (Höhe) und "width" (Weite). Unter Umständen, wenn diese Eigenschaften nicht explizit definiert werden, so existieren sie doch implizit durch den Kontext der umliegenden/Elternelementen und den Inhalt der Box selbst. In userem Fall ist der Inhalt ein kurzer Text ("Lorem ipsum").

Zunächst muss die Box über CSS angesprochen werden können. Wir vergeben ihr eine ID.

<div id="box">Lorem ipsum</div>

Nun kann die Box im CSS Dokument über die ID "box" angesprochen werden. Das Symbol # (Raute) adressiert eine ID.

div#box {
}

Jetzt wollen wir unserer Box eine explizite Größe definieren. Dazu vergeben wir die Attribute "height" und "width" im CSS Block der Box. Diese haben die folgenden Form:
[Attribute]:[Wert]

div#box {
width: 200px;
heigth: 100px;
}

Das Ergebnis:
Unsere Box ist blau eingefärbt, damit wir sie besser erkennen können.

Lorem ipsum

Der Innenabstand

Nachdem wir eine Box mit genauen Maßen erzeugt haben, arbeiten wir uns nun durch das CSS-Box-Model von Innen nach Außen.
Betrachten wir nun den inneren Abstand. Bei der Erstellung der Box fällt auf, dass der Inhalt der Box ("Lorem ipsum") direkt am linken Rand der Box beginnt. Das kann unter Umständen nicht sehr schön aussehen. Um den Abstand vom Rand der Box nach Innen zu definieren wird das Attribut padding ("Polster") verwendet.
Fügen wir unserer Box nun ein Padding Attribut hinzu. padding definiert den Abstand von allen Seitenrändern. Soll allerdings nur der Abstand vom oberen Boxrand definiert werden, so kann das Attribut padding-top verwendet werden.

div#box {
width: 200px;
heigth: 100px;
padding: 20px;
}

Das Ergebnis:
Die Schrift in der Box beginnt nun um 20 Pixel nach recht und unten verschoben. Der Padding-Bereich ist in violett markiert.

Lorem ipsum

Der Rahmen

Von Innen nach Außen gesehen, folgt nach dem Padding der Rahmen der Box. Der Rahmen wird über das Attribute border ("Grenze") angesprochen. Auch hier können, wie beim padding, die einzelnen Seitenränder mit beispielsweise border-top angesprochen werden.
Das Attribut border akzeptiert ein bis drei Attributwerte. Das sind Werte für die Farbe, die Dicke und die Art der Line. Art der Linie bedeutet, ob die Line beispielsweise gestrichelt oder durchgezogen ist.
Wir fügen unserer Box nun einen Rand hinzu, der eine durchgezogene Linie besitzt (solid), der 4 Pixel breit ist und die Farbe schwarz besitzt.

div#box {
width: 200px;
heigth: 100px;
padding: 20px;
border: 4px solid black;
}

Das Ergebnis:

Lorem ipsum

Der Außenabstand

Den äußersten Ring des CSS-Box-Models beschreibt der Außenabstand der Box zu anderen umgebenden Elementen. Den Außenabstand regelt das Attribut margin (Rand). Bei der margin lässt sich, wie bereits bei padding und border kennengelernt, eine einzelne Seite nach dem selben Muster ansprechen. Dabei kann beispielsweise das Attribut margin-top genutzt werden.

Wir wollen unserer Box nun einen Abstand nach Außen mit einer Weite von 10 Pixeln verpassen. Dazu fügen wir wieder das Attribute-Werte-Paar in den CSS-Block ein.

div#box {
width: 200px;
heigth: 100px;
padding: 20px;
border: 4px solid black;
margin: 10px;
}

Das Ergebnis:
Der Bereich des Margin ist in grün markiert.

Lorem ipsum

Zusammenfassung

Zusammenfassend lässt sich das CSS-Box-Model von Innen nach Außen zeigen:

Box(inhalt) hellblau
Padding violett
Border schwarz
Margin grün
