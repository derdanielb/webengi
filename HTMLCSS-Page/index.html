Demonstration der CSS-Box

Auf dieser Homepage wird die Zusammensetzung der CSS-Box in einem Step-by-Step-Verfahren ausführlich visualisiert.

Das Fundament

Als Basis haben wir eine einfache Box. Diese Box kann verschiedene Eigenschaften haben. Mindestens aber besitzt Sie die Eigenschaften "height" (Höhe) und "width" (Weite). Unter Umständen, wenn diese Eigenschaften nicht explizit definiert werden, so existieren sie doch implizit durch den Kontext der umliegenden/Elternelementen und den Inhalt der Box selbst. In userem Fall ist der Inhalt ein kurzer Text ("Lorem ipsum").

Zunächst muss die Box über CSS angesprochen werden können. Wir vergeben ihr eine ID.

<div id="box">Lorem ipsum</div>

Nun kann die Box im CSS Dokument über die ID "box" angesprochen werden. Das Symbol # (Raute) adressiert eine ID.

div#box {
}

Jetzt wollen wir unserer Box eine explizite Größe definieren. Dazu vergeben wir die Attribute "height" und "width" im CSS Block der Box. Diese haben die folgenden Form:
[Attribute]:[Wert]

div#box {
width: 200px;
heigth: 100px;
}

Das Ergebnis:
Unsere Box ist blau eingefärbt, damit wir sie besser erkennen können.

Lorem ipsum

Der Innenabstand

Nachdem wir eine Box mit genauen Maßen erzeugt haben, arbeiten wir uns nun durch das CSS-Box-Model von Innen nach Außen.
Betrachten wir nun den inneren Abstand. Bei der Erstellung der Box fällt auf, dass der Inhalt der Box ("Lorem ipsum") direkt am linken Rand der Box beginnt. Das kann unter Umständen nicht sehr schön aussehen. Um den Abstand vom Rand der Box nach Innen zu definieren wird das Attribut padding("Polster") verwendet.
Fügen wir unserer Box nun ein Padding Attribut hinzu. padding definiert den Abstand von allen Seitenrändern. Soll allerdings nur der Abstand vom oberen Boxrand definiert werden, so kann das Attribut padding-top verwendet werden.

div#box {
width: 200px;
heigth: 100px;
padding: 20px;
}

Das Ergebnis:
Die Schrift in der Box beginnt nun um 20 Pixel nach recht und unten verschoben. Der Padding-Bereich ist in violett markiert.

Lorem ipsum
